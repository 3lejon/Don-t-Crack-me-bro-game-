🎮 Don’t Cack Me Bro
Don’t Cack Me Bro is an action-packed rage platformer inspired by the golden era of classic Flash games.
Prepare yourself for brutally tight levels, ridiculous challenges, and an experience that will test your patience… in the best way possible.

🚀 About the Game
This game is designed to capture the nostalgic feel of old-school browser rage games, combining simple visuals with unforgiving gameplay mechanics. Every jump matters, every mistake costs you—and every victory feels earned.

⚡ Features

✏️ Hand-drawn graphics with a silly art style
🎨 Sprites created using Piskel
😈 Ridiculously unfair levels designed to push your limits
🧠 Tight Jumps requiring patience
🧗 Rage-inducing wall hops that demand skill and patience
🎵 Original self-made soundtrack


🎯 Gameplay
Expect:

near-perfect jumps
Trial-and-error progression
Sudden failure and instant retries
A whole lot of frustration… and satisfaction

If you enjoy games that challenge your skill and determination, this one’s for you.

🛠️ Tools Used

Piskel – for sprite creation
Unity as game engine
